## Overview

The Courses API enables structured learning paths by grouping concepts with mastery thresholds. Key features:

- Create courses with modules
- Add concepts with learning objectives
- Activate courses and enroll students
- Track individual student progress toward completion

**Authentication**: All course endpoints require authentication.

**Base path**: `/api/courses`

**Source**: `src/api/routes/courses.ts`

## Course Model

Courses have:
- **Status**: "draft" or "active"
- **Owner**: User who created the course
- **Organization**: Optional org scoping
- **Modules**: Optional grouping within the course
- **Concepts**: Linked concepts with mastery thresholds

**Source**: `src/api/db/schema.js` (courses table)

## Endpoints

### Create Course

Create a new course in draft status.

<CodeGroup>
```bash cURL
curl -X POST https://api.entendi.dev/api/courses \
  -H "x-api-key: <your-api-key>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Advanced React Patterns",
    "description": "Master compound components, render props, and hooks patterns",
    "orgId": "org_123"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.entendi.dev/api/courses', {
  method: 'POST',
  headers: {
    'x-api-key': '<your-api-key>',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Advanced React Patterns',
    description: 'Master compound components, render props, and hooks patterns',
    orgId: 'org_123'
  })
});
const course = await response.json();
```
</CodeGroup>

</ParamField>

#### Response

### Add Concept to Course

Link a concept to a course with optional learning objective and mastery threshold.

<CodeGroup>
```bash cURL
curl -X POST https://api.entendi.dev/api/courses/:id/concepts \
  -H "x-api-key: <your-api-key>" \
  -H "Content-Type: application/json" \
  -d '{
    "conceptId": "react-context-api",
    "moduleId": "module_xyz",
    "learningObjective": "Implement global state management using Context",
    "requiredMasteryThreshold": 0.8
  }'
```

```javascript JavaScript
const response = await fetch(
  `https://api.entendi.dev/api/courses/${courseId}/concepts`,
  {
    method: 'POST',
    headers: {
      'x-api-key': '<your-api-key>',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      conceptId: 'react-context-api',
      moduleId: 'module_xyz',
      learningObjective: 'Implement global state management using Context',
      requiredMasteryThreshold: 0.8
    })
  }
);
const courseConcept = await response.json();
```
</CodeGroup>

#### Path Parameters

<ParamField path="id" type="string" required>
  Course UUID
</ParamField>

#### Request Body

<ParamField body="conceptId" type="string" required>
  Concept identifier (must exist in concepts table)
</ParamField>

<ParamField body="moduleId" type="string">
  Module UUID (must belong to this course)
</ParamField>

<ParamField body="learningObjective" type="string">
  Specific learning objective (max 2000 characters)
</ParamField>

<ParamField body="requiredMasteryThreshold" type="number">
  Minimum mastery to consider complete (0-1 scale, default 0.7)
</ParamField>

#### Response

Returns the created course_concept join record.

**Status codes**:
- `201 Created` - Concept added successfully
- `400 Bad Request` - Validation error
- `404 Not Found` - Course, concept, or module not found

**Validation**:
- Verifies course exists
- Verifies concept exists
- Verifies module exists and belongs to course (if provided)

## Mastery Thresholds

Course concepts use a **required mastery threshold** (0-1 scale) to determine completion:

- **Default**: 0.7 (70% mastery)
- **Range**: 0.0 to 1.0
- **Interpretation**: Student must have ≥ threshold mastery to be considered complete

**Example thresholds**:
- `0.5` - Basic familiarity
- `0.7` - Solid understanding (default)
- `0.85` - Mastery
- `0.95` - Expert level

**Source**: `src/api/routes/courses.ts:27`, `src/schemas/types.ts:11-13`

## Zod Validation

Request bodies are validated using Zod schemas:

```typescript
const createCourseSchema = z.object({
  name: z.string().min(1).max(200),
  description: z.string().max(2000).optional(),
  orgId: z.string().max(200).optional(),
});

const addConceptSchema = z.object({
  conceptId: z.string().min(1).max(200),
  moduleId: z.string().max(200).optional(),
  learningObjective: z.string().max(2000).optional(),
  requiredMasteryThreshold: z.number().min(0).max(1).default(0.7),
});
```

**Validation errors** return `400 Bad Request` with detailed error information.

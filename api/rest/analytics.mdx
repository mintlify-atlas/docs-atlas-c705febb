## Overview

The Analytics API provides comprehensive learning insights including:

- Mastery timelines for individual concepts or aggregate progress
- Learning velocity metrics over rolling windows
- Activity heatmaps for calendar visualizations
- Detailed concept profiles with assessment history

**Authentication**: All analytics endpoints require authentication.

**Base path**: `/api/analytics`

**Source**: `src/api/routes/analytics.ts`

## Endpoints

### Get Concept Timeline

Retrieve mastery progression over time for a specific concept.

<CodeGroup>
```bash cURL
curl https://api.entendi.dev/api/analytics/timeline/:conceptId \
  -H "x-api-key: <your-api-key>"
```

```javascript JavaScript
const response = await fetch(
  `https://api.entendi.dev/api/analytics/timeline/${conceptId}`,
  {
    headers: { 'x-api-key': '<your-api-key>' }
  }
);
const data = await response.json();
```
</CodeGroup>

#### Path Parameters

<ParamField path="conceptId" type="string" required>
  Concept identifier (e.g., "react-hooks", "typescript-generics")
</ParamField>

#### Response

<ResponseField name="conceptId" type="string">
  The concept identifier
</ResponseField>

<ResponseField name="currentMastery" type="object | null">
  Current mastery state (null if never assessed)
  
  <Expandable title="properties">
    <ResponseField name="lower" type="number">
      Lower bound of 95% confidence interval (0-1 scale)
    </ResponseField>
    <ResponseField name="estimate" type="number">
      Point estimate of mastery (0-1 scale)
    </ResponseField>
    <ResponseField name="upper" type="number">
      Upper bound of 95% confidence interval (0-1 scale)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timeline" type="array">
  Assessment events in chronological order (up to 200 events)
  
  <Expandable title="event object">
    <ResponseField name="eventId" type="string">
      Assessment event ID
    </ResponseField>
    <ResponseField name="timestamp" type="string">
      ISO timestamp of assessment
    </ResponseField>
    <ResponseField name="mastery" type="object">
      Mastery range at this point (lower, estimate, upper)
    </ResponseField>
    <ResponseField name="eventType" type="string">
      Type: "probe", "tutor_phase1", "tutor_phase4", "implicit"
    </ResponseField>
    <ResponseField name="rubricScore" type="number">
      Score: 0 (fail), 1 (partial), 2 (good), 3 (excellent)
    </ResponseField>
    <ResponseField name="integrityScore" type="number | null">
      Confidence in assessment validity (0-1)
    </ResponseField>
  </Expandable>
</ResponseField>

## Mastery Range Format

All mastery values use a confidence interval representation:

```json
{
  "lower": 0.42,
  "estimate": 0.63,
  "upper": 0.81
}
```

- **estimate**: Point estimate of mastery (sigmoid of μ)
- **lower/upper**: 95% confidence interval bounds
- **Scale**: 0-1 (0% to 100% mastery)

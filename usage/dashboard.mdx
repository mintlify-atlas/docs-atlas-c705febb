## Overview

The Entendi dashboard at [entendi.dev](https://entendi.dev) provides a comprehensive view of your learning progress, concept mastery, and assessment history.

## Main Sections

The dashboard is organized into tabs that track different aspects of your comprehension journey:

### Overview

Your landing page shows:

- **Strongest Domains**: Concepts where you've demonstrated highest mastery
- **Needs Attention**: Concepts that may need review or practice
- **Recent Activity**: Timeline of your latest assessments and probes

This view gives you a quick snapshot of your current learning state.

### Analytics

Detailed visualizations of your learning patterns:

- **Activity Heatmap**: Assessment frequency by day over the past year
- **Learning Velocity**: Cumulative mastery gain over time
  - Toggle between different time windows to see your progress trajectory
- **Domain Strengths**: Radar chart comparing mastery levels across knowledge domains
- **Review Needed**: List of concepts predicted to decay based on spacing algorithms

### Concepts

Browse and filter all concepts you've encountered:

- View total assessed concepts with mastery scores
- Filter by domain or mastery level
- Click any concept to see:
  - Current mastery probability
  - Assessment count and history
  - Related concepts (prerequisites and dependencies)
  - Uncertainty metrics (mu, sigma)

### Integrity

Monitor assessment authenticity:

- **Integrity Trend**: Response integrity scores over time
- **Flagged Responses**: Assessments that triggered behavioral anomaly detection
  - High typing speed
  - Unusual formatting patterns
  - Style drift from baseline

Integrity monitoring helps ensure genuine comprehension rather than AI-assisted responses.

### Codebases

Track learning within specific repositories:

- View enrolled codebases and associated concepts
- See mastery progress for concepts in each codebase
- Organization members can view team-wide codebase progress

### Syllabi

Structured learning paths:

- Browse available syllabi in your organization
- Track progress through curriculum sequences
- See prerequisite chains and recommended learning order

### Organization

*Available for organization members*

Manage team learning:

- View member mastery overview
- Access leaderboards and rankings
- Review aggregate analytics
- Configure enforcement levels
- Manage role-based access

See [Organizations](/usage/organizations) for details.

### Settings

Account and preference management:

- Update profile information
- Manage API keys
- Configure notification preferences
- View subscription details
- Delete account

## Navigation Tips

- Use keyboard shortcuts to switch between tabs
- The sidebar provides quick filters for domain-specific views
- All charts are interactive—hover for detailed tooltips
- Export data via the API for custom analysis

## Authentication States

- **Logged Out**: Landing page with product overview
- **Logged In**: Full dashboard with personalized data
- **No Activity**: Dashboard shows onboarding prompts until first assessment

## Social Login

The dashboard supports authentication via:

- **GitHub OAuth** (if configured)
- **Google OAuth** (if configured)
- **Email/Password**

See [Login](/usage/login) for authentication workflows.
